Groq API Key Setup Guide

Follow these simple steps to get your free API key for AI grading and transcription.

Step 1: Visit Groq Console
Go to the Groq Cloud Console.
Open console.groq.com →
Step 2: Sign in with Google
On the sign-in page, click the "Continue with Google" button to use your Gmail account for instant SSO setup.
Step 3: Go to API Keys
Once logged in, locate the "API Keys" tab in the left-hand sidebar menu.
Go directly to API Keys →
Step 4: Create Your Key
Click the "Create API Key" button. Give your key a name (for example: "JLPT-App") and click create.
Step 5: Copy and Paste
Copy the generated key immediately (it starts with gsk_). Go back to the app, paste it into the Groq API Key field, and click "Save Key".
